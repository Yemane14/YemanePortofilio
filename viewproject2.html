Project

Adimera Ecommerce Website

As a full-stack developer at Adimera Technologies PLC, I designed and developed an ecommerce website using Spring Boot Java, PostgreSQL, and Next.js. This ecommerce website allows customers to purchase products using Abyssinia Bank online payment methods.

Key features of the ecommerce website:

  • User-friendly interface with a clear navigation menu, search function, and product categories
  • Secure checkout process with Abyssinia Bank online payment
  • Integration with a third-party shipping service for timely delivery
  • Administration dashboard to manage products, orders, and customer information
  • Customizable email notifications for order status updates

I developed this website using modern web technologies, including Node.js, PostgreSQL, Next.js, and Material-UI. Hosted on a cloud platform, it incorporates robust security measures to safeguard customer data and prevent fraud.

This ecommerce platform has been instrumental in increasing sales and expanding the business's customer base. I am proud of my work and look forward to contributing innovative solutions to other businesses and organizations.
