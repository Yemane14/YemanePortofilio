Adimera Ecommerce Website

As a full-stack developer at Adimera Technologies PLC, I designed and developed an ecommerce website using Node.js, PostgreSQL, and Next.js. This ecommerce website allows customers to purchase products using Abyssinia bank online payment.

Some of the key features of the ecommerce website include:

  • User-friendly interface with a clear navigation menu, search function, and product categories
  • Secure checkout process with Abyssinia bank online payment
  • Integration with a third-party shipping service to ensure timely delivery of products
  • Administration dashboard to manage products, orders, and customer information
  • Customizable email notifications for order status updates

I developed this ecommerce website using a range of modern web development technologies, including Node.js, PostgreSQL, Next.js, and Material-UI. The site is hosted on a cloud platform with robust security measures to protect customer data and prevent fraud.

The ecommerce website has been a tremendous success for the business, helping to increase sales and reach new customers. You can view the website at Adimera Ecommerce.
